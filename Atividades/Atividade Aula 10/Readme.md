🕰️ 1. Linha do Tempo
Principais marcos da evolução das redes:
Década de 1960 – Mainframes
Computadores grandes e centralizados
Acesso por terminais
1969 – ARPANET
Primeira rede de computadores
Projeto militar dos EUA
Década de 1970 – Criação do TCP/IP
Base da comunicação entre redes
1983 – Padronização do TCP/IP
Surgimento da internet moderna
Década de 1990 – Internet Comercial
Popularização global
Surgimento da Web
2000 até hoje – Redes Móveis
Wi-Fi, smartphones
4G e 5G
🔌 2. Elementos da Rede
Esquema:
Emissor → Meio de Transmissão → Receptor
          ↳ Protocolo controla a comunicação
Explicação:
Emissor
Dispositivo que envia a informação
Ex: celular, computador
Receptor
Dispositivo que recebe a informação
Ex: servidor, notebook
Meio de Transmissão
Caminho por onde os dados passam
Ex: Wi-Fi, cabo de rede, fibra óptica
Protocolo
Regras da comunicação
Ex: TCP/IP, HTTP
🌐 3. Classificação de Redes
Tipo	Nome	Alcance	Exemplo
PAN	Personal Area Network	Poucos metros	Bluetooth (celular + fone)
LAN	Local Area Network	Casa/escola	Wi-Fi doméstico
MAN	Metropolitan Area Network	Cidade	Rede de universidade
WAN	Wide Area Network	Global	Internet
