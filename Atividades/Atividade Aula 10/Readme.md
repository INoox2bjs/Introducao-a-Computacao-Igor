## Descrição

Este trabalho apresenta os principais conceitos sobre redes de computadores, incluindo:

* A evolução histórica das redes
* Os elementos fundamentais de uma rede
* A classificação das redes conforme sua abrangência

Os conteúdos foram desenvolvidos com base nas aulas e no livro de Tanenbaum.
