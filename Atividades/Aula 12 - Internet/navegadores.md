# Navegadores

## Função dos navegadores

Os navegadores são programas responsáveis por acessar sites e interpretar o conteúdo da web.

Eles transformam o código em páginas visuais e interativas, processando:

- HTML → estrutura da página  
- CSS → estilo (cores, layouts, fontes)  
- JavaScript → interatividade (botões, animações, ações)

Em resumo: o navegador “lê o código” e mostra a página pronta para o usuário.

---

## Motores de renderização

Os motores de renderização são partes internas do navegador responsáveis por exibir o conteúdo na tela.

| 🔧 Motor | 🌐 Navegador |
|---|---|
| Blink | Chrome / Edge |
| Gecko | Firefox |
| WebKit | Safari |

---

## Resumo

```text
HTML + CSS + JavaScript
          ↓
     🖥️ Navegador
          ↓
   📄 Página exibida
```
