# Cliente e Servidor

```text
        REQUISIÇÃO
┌──────────────┐  ─────────────►  ┌──────────────┐
│   CLIENTE    │                 │   SERVIDOR   │
│  Navegador   │  ◄─────────────  │     Site     │
└──────────────┘     RESPOSTA     └──────────────┘
```

📌 O cliente faz a requisição e o servidor responde com os dados (página, imagem, etc.).

# Endereço IP

## IPv4

```text
192.168.0.1
```

Formato mais comum, com 4 blocos de números.

---

## IPv6

```text
2001:0db8:85a3:0000:0000:8a2e:0370:7334
```

Novo formato criado para suportar mais dispositivos na Internet.
